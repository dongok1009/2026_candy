# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-24

### Added
- **1분봉 정밀 백테스트**: 5분봉 내의 미세한 움직임을 1분 단위로 쪼개어 익절/손절 시점을 초 단위까지 추적하는 로직 구현.
- **현실적 비용 반영**: 진입 시 시장가 수수료(0.05%), 종료 시 지정가 수수료(0.02%) 및 8시간마다 발생하는 펀딩피(0.01%) 자동 계산 로직 적용.
- **수수료 공제 후 6% 순익 조건**: 모든 거래 비용을 제외하고도 순수한 수익이 6%가 달성되도록 익절가를 자동 보정하는 기능 추가.
- **일봉 필터 강화**: 일봉 StochRSI 지표의 K선과 D선 격차가 2를 초과할 때만 진입하는 신호 필터링 추가로 안정성 확보.
- **24시간제 시간 통합**: 백테스트 CSV 결과와 대시보드 내 시간 입력 및 출력을 24시간제(YYYY-MM-DD HH:mm:ss)로 통일. '00시'를 포함한 정밀 시간 입력 및 조회 지원.

### Changed
- 백테스트 스뮬레이션 방식을 '중복 진입 차단형' 직렬 구조로 개편하여 복리 계산 및 비용 정산의 정확도 향상.
- 대시보드(React)의 '과거 모드' 검색 필드 수정 (step="1" 추가로 초 단위 입력 활성화).
