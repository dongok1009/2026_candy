# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2026-03-25

### Added
- **1일봉 MACD-Signal 필터**: 1일봉 MACD 히스토그램(|MACD - Signal|)의 임계값에 따른 진입 제한 조건 추가.
- **Interactive MACD Filter**: 대시보드 하단에서 1일봉의 MACD 히스토그램 필터를 활성화하고 임계값을 실시간으로 조정할 수 있는 기능 탑재.

### Changed
- **UI 개선**: 설정창의 숫자 입력 필드 너비를 확장하여 시인성 확보.

## [1.2.0] - 2026-03-25

### Optimized
- **수익률 극대화**: 익절 목표를 **4.00%(Net)**로 상향 최적화하여 복리 효율성 약 37% 향상.
- **연속 재진입(Re-entry)**: 익절 직후에도 지표 조건 만족 시 즉시 재진입하여 파동의 모든 수익을 취하는 로직 정식 탑재.
- **백테스트 데이터 확보**: 8개월간 **1,885% 수익(승률 89.1%)** 기록 기반의 5배(5x) 레버리지 세팅 확정.

## [1.0.0] - 2026-03-25

### Official Release (Stable)
- **Wait-on-Close (봉 마감 기반 정체 신호)**: 5m, 1h, 1d 차트의 실시간 흔들림을 배제하고, '직전 확정 캔들'의 지표만을 분석하여 승률 극대화.
- **백테스트 정밀화**: 익절 3%(Net), 손절 15%, 5배 레버리지 기반 시뮬레이션에서 8개월간 **1,221% 수익(승률 92.1%)** 달성.
- **UI 접근성 개선**: 24시간제 검색 지원 및 하이브리드 시간 입력기(마우스+키보드) 최적화.
- **시장가/지정가 수수료**: 매수(시장가 0.05%), 매도(지정가 0.02%) 및 펀딩 피가 실시간 차감되는 매매 비서 고도화.
